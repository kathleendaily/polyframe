# Polyframe
React app for converting images to low-polygon

<img
  alt="Screenshot of the Polyframe app"
  src="./assets/images/demo-screenshot.png"
  width="100%"
/>

## Running Locally

Clone the project and run

```bash
$ npm install
```

```bash
$ npm start
```

This will open the app on http://localhost:8080
